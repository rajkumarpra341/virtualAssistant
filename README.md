# 🤖 Virtual Assistant (MERN Stack)

## 📌 Project Overview

Virtual Assistant is a full-stack AI-powered web application built using the MERN Stack. The application allows users to create an account, log in securely, customize their personal AI assistant, and interact with it through both text and voice commands.

The assistant can understand user queries, listen to voice input, and generate intelligent responses similar to ChatGPT, providing a natural conversational experience.

This project showcases full-stack development, authentication, AI integration, voice recognition, and user personalization features.
